# 职聊凭证 AI 演示脚本

## 30 秒介绍

职聊凭证 AI 面向求职者在招聘沟通中的信息留痕场景。它可以把零散的 HR 聊天记录整理成结构化承诺清单，自动标出薪资、地点、试用期、社保、加班、试岗等关键信息，并生成一份可发送给 HR 二次确认的沟通确认单。

## 演示流程

1. 打开首页
2. 点击样例 1，展示聊天记录自动填充
3. 点击分析记录
4. 展示已确认事项、未确认事项、风险提醒、建议追问
5. 展示底部招聘沟通确认单
6. 点击导出 Markdown 或导出 JSON
7. 切到 API 标签页，展示可接入 OpenAI Responses、DeepSeek、New API
8. 点击检测接入和检测模型，展示模型列表和模型选择
9. 切到截图标签页，说明可上传聊天截图进行 OCR

## 评委关注点回答

### 为什么有价值

招聘沟通大量发生在聊天工具里，求职者很容易遗漏口头承诺和模糊表述。这个系统把聊天内容转成可确认的结构化凭证，帮助双方在面试或入职前减少理解偏差。

### AI 做了什么

AI 负责三件事：从聊天中抽取承诺，识别未确认和模糊事项，生成追问话术和确认单。

### 为什么适合人才服务

它可以接入招聘平台、校园就业系统、人力服务机构和求职社群，提升招聘沟通透明度。

### 后续如何商业化

可以做成求职者工具、招聘平台插件、校园就业服务系统组件，也可以给 HR 端提供沟通合规检查。

## 推荐演示样例

使用页面内置样例 1。它包含薪资、地点、试用期、工作时间、社保和报销，能展示完整抽取效果。

## 成功标准

1. 能从聊天记录提取至少 5 个已确认事项
2. 能识别至少 2 个未确认事项
3. 能生成追问话术
4. 能导出确认单
5. 能展示真实模型接入能力
